# 🤖 ШІ-Помічник Telegram (Лабораторна Робота)

Асинхронний Telegram-бот на базі штучного інтелекту OpenAI, розроблений строго за офіційним ТЗ курсу.

## 🚀 Головні фішки та архітектура
- **Повна асинхронність:** Серце бота переведено на сучасний `AsyncOpenAI` для фонової генерації відповідей без блокування потоків програми.
- **Оптимізація витрат:** Використовується спритна та економічна модель `gpt-4o-mini`.
- **Безпека даних:** Усі приватні токени надійно ізольовані у файлі `.env` та захищені за допомогою `.gitignore`.
- **Модульний дизайн:** Усі промпти та системні привітання автономно завантажуються з файлової системи.

## 🛠️ Функціонал за ТЗ
1. `/random` — Генерація унікальних цікавих фактів від ШІ з динамічними кнопками.
2. `/gpt` — Прямий діалог з ШІ-асистентом у режимі ехо-бота.
3. `/talk` — Інтерактивний діалог із видатними особистостями (Кобейн, Ніцше, Гокінг та ін.).
4. `/quiz` — Інтелектуальна покрокова вікторина з розумним підрахунком балів та фінальним відсотком вінрейту.
