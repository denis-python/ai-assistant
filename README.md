# AI Assistant Telegram Bot

Багатофункціональний Telegram-бот на базі штучного інтелекту, що надає доступ до ChatGPT через різні режими роботи.

## Основні режими:
- **🧠 Випадковий факт**: Отримання цікавих фактів.
- **🤖 Чат з ChatGPT**: Класичний діалог із ШІ.
- **👤 Особистості**: Спілкування з відомими людьми (Курт Кобейн, Стівен Гокінг та ін.).
- **❓ ШІ-Вікторина**: Тематичні квізи (Python, Математика, Біологія) з підрахунком балів.
- **🌐 Перекладач**: Автоматичний переклад тексту.
- **🎬 Кінокритик**: Персоналізований підбір фільмів.

## Налаштування:
1. Клонуйте репозиторій.
2. Створіть файл `.env` та додайте туди токени:OPENAI_TOKEN та TELEGRAM_TOKEN
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть бота: `python bot.py`
